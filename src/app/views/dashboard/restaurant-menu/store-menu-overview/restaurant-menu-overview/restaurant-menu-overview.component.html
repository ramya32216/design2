Explore the tabs above to set up your menu. You can start building in any order, however before your menu appears here (or in the app) you must link a menu to a category and that category to some items.

  • Menus

    Add new menus (e.g. breakfast, lunch and dinner).

  • Categories

    Add categories and select which menus they appear in.

  • Items

    Add items and select which categories they appear in.

  • Modifier groups

    Add options for your items (e.g. extras, sauce and size).
